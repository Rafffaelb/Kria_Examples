
# I2C Communication Example
# =========================

This project provides a basic example of establishing I2C communication on the Kria KR260.

## Overview
*   **Goal**: Interface with external peripherals (e.g., sensors, displays) using the I2C protocol.
*   **Key Components**: AXI IIC, Zynq PS I2C.

## Usage
1.  Open the Vivado project in this directory (if applicable) or run the build scripts.
2.  Review the driver code to understand initialization and read/write sequences.
